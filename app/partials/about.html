About Awesome Interface

Awesome interface started out as a practice python project. Because most of the awesome lists follow the format provided by awesome manifesto, it seemed like a good idea to target all the awesome lists, parse data from them and and dump it to a JSON object.

Since I already had the JSON, might as well us it to render the lists in a user friendly way for the developers in the community! The python scripts used to scrap the data from awesome lists were init.py and parse.py.


Future Considerations

If the community wants, I could expand on this project by perhaps creating a login system where user can create, review and vote custom lists. Popular custom lists could be something like a guide towards learning certain topic or maybe a list of concert events of certain celebrity. Suggestions are appreciated.
